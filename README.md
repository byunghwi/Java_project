# Convienience Project

---------------------
__1. 개요__

 상품판매, 재고관리, 정산, 이벤트, 근태관리 기능이 있는 편의점 포스기 프로그램
 
------------------------
 __2. 개발환경__

>- OS : Window 10 
>- 개발언어 : JAVA 
>- DB : Oracle 
>- Tool : Eclipse (jdk-14) 
>- 외부 라이브러리 :  jdbc(ojdbc8.jar) / jfreechart(jcommon-1.0.23.jar, jfreechart-1.0.19.jar)
